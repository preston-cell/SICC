## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Can't login | Check credentials, clear cookies, try incognito |
| Slow responses | Check internet, try backup URL |
| Upload fails | Try different file, check file size |
| AI gives unexpected answer | Acknowledge, rephrase, continue |
