# Utah

## Key Differences from Other States
- No state estate tax
- No state inheritance tax
- Common law property state
- Transfer on Death Deed available
- Uniform Probate Code adopted
- Holographic wills recognized
- Moderate homestead protection ($45,800)

## Will Requirements
**Statutory Basis**: Utah Code Title 75 (Uniform Probate Code)

### Execution Requirements (Utah Code § 75-2-502)
- Testator must be **18+ years old**
- Must be of **sound mind**
- Must be **in writing**
- Must be **signed by testator** (or by another at direction in presence)
- Must be signed by at least **2 competent witnesses**
- Witnesses must sign within reasonable time

### Self-Proving Affidavit (Utah Code § 75-2-504)
- Available and recommended
- Testator and witnesses sign affidavit before notary
- Creates presumption of proper execution
- Avoids need to locate witnesses at probate

### Holographic Wills (Utah Code § 75-2-502)
- **Recognized** in Utah
- Material provisions must be in testator's handwriting
- Must be **signed by testator**
- No witness requirement
- Date recommended but not required

### Witness Requirements
- Must be competent witnesses
- Must sign within reasonable time of witnessing
- Should not be beneficiaries (best practice)
- Interested witness does not automatically invalidate

## Trust Requirements
**Statutory Basis**: Utah Code Title 75, Chapter 7 (Utah Uniform Trust Code)

### Trust Creation (Utah Code § 75-7-402)
- Utah Uniform Trust Code adopted
- Must have trust property
- Must have ascertainable beneficiaries (or valid purpose)
- Settlor must have capacity
- Writing required for real property

### Trust Administration
- Utah UTC governs
- Prudent Investor Rule applies
- Trustee has duties of loyalty, impartiality
- Accounting to beneficiaries required
- Notice upon settlor's death

### Revocable Trusts (Utah Code § 75-7-602)
- Common for probate avoidance
- Settlor retains control during lifetime
- Presumed revocable unless stated otherwise
- Becomes irrevocable at settlor's death

### Directed Trusts (Utah Code § 75-7-906)
- Utah allows directed trusts
- Trust advisor can direct trustee
- Trustee liability limited when following directions

### Trust Decanting
- Available under Utah law
- Allows modification of irrevocable trusts
- Trustee discretionary power required

## Power of Attorney
**Statutory Basis**: Utah Code Title 75, Chapter 9 (Uniform Power of Attorney Act)

### Execution Requirements (Utah Code § 75-9-105)
- Must be **signed by principal** (or at direction)
- Must be **notarized** (acknowledged)
- Witnesses not required but recommended
- Statutory form available

### Durability (Utah Code § 75-9-104)
- Durable by default unless document states otherwise
- Non-durable POA terminates upon incapacity
- Springing POA available

### Statutory Form (Utah Code § 75-9-301)
- Utah provides statutory form (recommended)
- Other forms valid if properly executed
- Third parties must accept valid POA

### Agent Duties
- Fiduciary duty to principal
- Must act in principal's best interest
- Must keep records
- Must avoid conflicts of interest

### Hot Powers (Utah Code § 75-9-201)
- Certain powers require express grant:
  - Create, amend, revoke trusts
  - Make gifts
  - Create/change beneficiary designations
  - Create/change rights of survivorship

### Third Party Acceptance (Utah Code § 75-9-120)
- Third parties must accept valid POA
- May request certification
- Refusal may result in liability

## Healthcare Directive
**Statutory Basis**: Utah Code Title 75, Chapter 2a (Advance Health Care Directive Act)

### Types Available
- Advance Health Care Directive (combined form)
- Living Will (Part of directive)
- Health Care Agent (Part of directive)

### Execution Requirements (Utah Code § 75-2a-107)
- Must be **in writing**
- Must be **signed by declarant** (or at direction)
- Must be **witnessed by 1 adult** OR **notarized**
- Witness cannot be:
  - Healthcare provider
  - Agent named in directive

### Agent Authority
- Make healthcare decisions when principal incapacitated
- Access medical records
- Consent to or refuse treatment
- Arrange care and placement
- Make anatomical gift decisions

### Utah POLST
- Available in Utah
- Physician Orders for Life-Sustaining Treatment
- Medical orders for seriously ill patients
- Completed with healthcare provider
- Portable across healthcare settings

### Default Surrogate (Utah Code § 75-2a-110)
- If no healthcare directive:
  - Guardian
  - Spouse
  - Adult child
  - Parent
  - Adult sibling
  - Any adult relative

## Estate and Inheritance Tax
- **No state estate tax**
- **No state inheritance tax**
- **No state gift tax**
- Only federal estate tax applies if estate exceeds federal threshold
- 2025 federal exemption: $13.99 million per person

### Planning Implications
- No state death taxes simplifies planning
- Focus on federal estate tax for large estates
- Probate avoidance for privacy and efficiency
- Transfer on Death tools widely used

## Medicaid Planning (Utah Medicaid)

### Look-Back Period
- **60 months** (5 years)

### Asset Limits (2025)
- Individual: $2,000
- Married (community spouse): CSRA up to federal maximum (~$154,140)
- CSRA minimum: approximately $31,584

### Income Limits
- Approximately $2,829/month for individuals
- Qualified Income Trust (Miller Trust) available if over limit

### Exempt Assets
- Primary residence (equity limit ~$713,000)
- One vehicle
- Personal belongings and household goods
- Prepaid irrevocable burial contracts
- Term life insurance
- Whole life with face value under $1,500

### Utah-Specific Programs
- **Utah Medicaid** administered by Utah Department of Health
- **New Choices Waiver** for home-based services
- **Aging Waiver** program
- Estate recovery program active

### Spousal Protections
- Community spouse resource allowance (CSRA)
- Minimum monthly maintenance needs allowance (MMMNA)
- Court order for increased CSRA available

### Penalty Divisor
- Approximately $7,000-8,000/month (verify current rate)

## Homestead Protection
**Utah Code § 78B-5-503**

### Exemption Amount
- Up to **$45,800** in equity protected (indexed for inflation)
- Adjusted periodically
- Applies to dwelling and land

### Requirements
- Must be **principal residence**
- Must be owner
- Automatic protection (no filing required)
- Declaration can strengthen protection

### Scope of Protection
- Protected from most judgment creditors
- Does NOT protect against:
  - Mortgages and deeds of trust
  - Property tax liens
  - Mechanic's liens
  - Child support/alimony

### Moderate Protection
- Indexed for inflation
- Consider supplemental asset protection strategies
- Trust planning for larger equity

## Transfer on Death Deed
**Utah Code § 75-6-401 et seq.**

### Features
- Transfer real property on death **without probate**
- Revocable during owner's lifetime
- Does not affect owner's rights during lifetime
- Uniform Real Property Transfer on Death Act

### Requirements
- Must be **in writing**
- Must be **signed** by owner
- Must be **notarized** (acknowledged)
- Must be **recorded before death** with county recorder
- Must designate beneficiary

### Advantages
- Avoids probate for real property
- Simple and cost-effective
- Step-up in basis at death
- Revocable during lifetime

### Limitations
- Subject to decedent's debts
- Subject to claims of creditors
- Only for Utah real property

### Revocation
- Record revocation instrument
- Subsequent TOD deed supersedes prior
- Conveyance during life revokes
- Not revoked by will

## Probate Procedures

### Informal Probate (Utah Code § 75-3-301)
- Available for uncontested estates
- Minimal court involvement
- Personal representative appointed by register
- Most common form (UPC state)

### Formal Probate (Utah Code § 75-3-401)
- Required if will contested
- Full court supervision
- More expensive and time-consuming

### Small Estate Procedures (Utah Code § 75-3-1201)
- **Summary administration** for estates under **$100,000**
- Simplified procedures
- Affidavit procedure for smaller personal property

### Timeline
- Informal probate: 6-12 months typical
- Formal probate: 12-24 months
- Small estate: 2-4 months
- Creditor claim period: 3 months from notice

### Court Jurisdiction
- **District Court** handles probate
- County where decedent resided has jurisdiction

## Safe Deposit Box Rules

### Access After Death
- Joint renter has immediate access
- Without joint renter: letters testamentary from district court required
- Bank may permit examination with death certificate to locate will
- Personal representative needed for full access

### Joint Ownership
- Surviving joint renter retains full access
- No automatic freeze upon death
- Contents accessible without court involvement

### Best Practices
- Original will should NOT be stored in safe deposit box
- No state estate or inheritance tax simplifies planning
- Document box contents with estate plan
- Inform personal representative of box location

## State Charitable Incentives

### Utah Educational Savings Plan Credit
- Credit for contributions to Utah 529 plans
- 4.65% credit (equal to state income tax rate)
- Up to $2,220 credit per beneficiary (2025)
- Supports education funding

### Voluntary Contributions via Tax Return
- Checkoff options for various Utah charities
- Convenient giving mechanism

### Estate Planning Implications
- 529 plan contributions receive state credit
- Federal charitable deductions available
- No specific estate/gift tax benefits (no state death tax)
- Consider charitable planning for federal tax benefits
