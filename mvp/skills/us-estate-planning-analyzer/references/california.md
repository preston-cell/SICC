# California

## Key Differences from Other States
- **Community property state** with strict rules
- No state estate tax
- No state inheritance tax
- **Unique Medi-Cal look-back**: historically 30 months (changes in flux - verify current rules)
- Strong tenant protections and homestead exemptions
- **Complex and expensive probate system**
- Transfer on Death Deed available
- Proposition 13 and 19 affect inherited real property taxes

## Community Property Rules

### Classification (Family Code § 760)
- All property acquired during marriage is community property
- Separate property includes: property owned before marriage, gifts, inheritance
- Income from separate property is **separate property** (American Rule)
- Transmutation requires express written declaration
- Quasi-community property rules for property acquired out of state

### Transmutation Rules (Family Code § 850-853)
- Written declaration required to change character of property
- Must be signed by spouse whose interest is adversely affected
- Gift between spouses must be express
- Oral agreements insufficient

### Quasi-Community Property (Probate Code § 66)
- Property acquired while living outside California
- Treated as community property for death/divorce
- Applies if would have been CP in California
- Important for couples who moved to California

### Estate Planning Implications
- Each spouse owns 50% of community property
- **Full step-up in basis** for both halves at first spouse's death (IRC § 1014(b)(6))
- Each spouse can dispose of their 50% by will
- Cannot disinherit spouse without valid waiver

### Commingling and Tracing
- Commingled property presumed community
- Tracing can establish separate character
- Clear records essential
- Family expense presumption applies

## Will Requirements
**Statutory Basis**: California Probate Code §§ 6100-6113

### Execution Requirements (Prob. Code § 6110)
- Testator must be **18+ years old**
- Must be of **sound mind**
- Must be **in writing**
- Must be **signed by testator** (or by someone at testator's direction in presence)
- Must be signed by at least **2 witnesses**
- Witnesses must understand document is testator's will
- Witnesses must sign during testator's lifetime

### Witness Requirements
- Witnesses must be generally competent
- Interested witnesses allowed but creates rebuttable presumption of undue influence
- Should not be beneficiaries (best practice)

### Self-Proving Affidavit
- Not traditionally used in California
- Cal. Prob. Code § 8220 allows affidavit in lieu of testimony
- Witnesses typically appear at probate or submit declaration

### Holographic Wills (Prob. Code § 6111)
- **Recognized** if material provisions in testator's handwriting and signed
- **No witness requirement**
- Date recommended but not required
- Testamentary intent must be clear

### California Statutory Will (Prob. Code § 6240)
- Fill-in-the-blank statutory will form available
- Valid if properly completed
- Limited flexibility
- Appropriate for simple estates

## Trust Requirements
**Statutory Basis**: Probate Code §§ 15000-18201

### Trust Creation (Prob. Code § 15200)
- Must be in writing for real property
- No witness requirement for revocable trusts
- Must have trust property
- Must have ascertainable beneficiaries

### Community Property Trusts
- Property transferred to trust retains community property character
- Critical for maintaining step-up in basis benefits
- Must be drafted to preserve CP status
- Both spouses should be co-trustees and co-beneficiaries

### Trust Administration
- Probate Code Part 4 governs
- Trustee has duty to provide accountings
- 120-day notification requirement upon death (Prob. Code § 16061.7)
- Prudent Investor Rule applies

### Trust Contests
- 120-day statute of limitations from notification
- Grounds: lack of capacity, undue influence, fraud

## Power of Attorney
**Statutory Basis**: Probate Code §§ 4000-4545

### Execution Requirements (Prob. Code § 4121)
- Must be **signed by principal**
- Must be **witnessed by 2 adults** OR **notarized**
- If signing real property documents, **must be notarized**
- Durable if includes durability language

### Statutory Form (Prob. Code § 4401)
- California has statutory form (strongly recommended)
- Other forms valid but may face resistance from third parties
- Financial institutions may have their own forms
- Statutory form widely recognized

### Durability (Prob. Code § 4124)
- Must include: "This power of attorney shall not be affected by subsequent incapacity of the principal"
- Non-durable POA terminates upon incapacity
- Springing POA available

### Third Party Acceptance
- Financial institutions must accept valid POA
- May require certification (Prob. Code § 4402)
- Refusal may result in liability

## Healthcare Directive
**Statutory Basis**: Probate Code §§ 4700-4701 (Advance Health Care Directive)

### Advance Health Care Directive Form
- Combines healthcare proxy and living will
- Must be **witnessed by 2 adults** OR **notarized**
- At least one witness cannot be healthcare provider
- At least one witness cannot be operator of community care facility

### Witness Requirements (Prob. Code § 4701)
- Two adult witnesses required (unless notarized)
- One witness cannot be:
  - Healthcare provider
  - Employee of healthcare provider
  - Operator/employee of residential care facility where patient resides
- Both witnesses should not be related or entitled to estate (recommended)

### Agent Authority
- Make healthcare decisions when principal incapacitated
- Access medical records (HIPAA authorization included)
- Consent to or refuse treatment
- Arrange care and placement
- Anatomical gift decisions

### POLST (Physician Orders for Life-Sustaining Treatment)
- Available in California
- Medical orders, not advance directive
- For seriously ill patients
- Completed with healthcare provider

## Estate and Inheritance Tax
- **No state estate tax** (repealed 1982)
- **No state inheritance tax**
- **No state gift tax**
- Only federal estate tax applies if estate exceeds federal threshold
- 2025 federal exemption: $13.99 million per person

## Medicaid Planning (Medi-Cal)

### Look-Back Period
- **Historically 30 months** (significantly shorter than other states)
- **Important**: California rules have been in flux
- AB 133 (2021) eliminated asset test for many programs
- Verify current rules - significant changes occurred 2022-2024

### Asset Limits (Verify Current Rules)
- Rules have changed significantly
- Historically $2,000 individual, now varies by program
- CSRA rules apply for married couples
- No asset limit for some Medi-Cal programs as of 2024

### Exempt Assets (Traditional)
- Primary residence (equity limit ~$1,071,000 for California - higher than most states)
- One vehicle
- Personal belongings and household goods
- Prepaid irrevocable burial contracts
- Term life insurance

### California-Specific Considerations
- **Medi-Cal** has unique income and asset rules
- Share of cost requirements for some beneficiaries
- **Higher cost of living** affects nursing home costs significantly
- AB 133 made significant changes - verify current rules
- Estate recovery program active (MERP)

### Community Property and Medi-Cal
- Community property rules complicate Medi-Cal planning
- CSRA protects community spouse's share
- Proper characterization critical
- All community property considered available

### Penalty Divisor
- Approximately $12,000-14,000/month (verify current - varies by county)

## Homestead Protection
**California Code of Civil Procedure § 704.710 et seq.**

### Exemption Amounts (2025 - indexed for inflation)
- **Automatic homestead**: $300,000 - $600,000+ (varies by county median home price)
- Minimum: $300,000
- Maximum: $600,000
- Adjusted annually based on county median sale price

### Declared Homestead
- Same dollar amounts as automatic
- Provides additional protection for sale proceeds
- Must be recorded with county recorder
- Protects proceeds for 6 months after sale

### Requirements
- Must be principal dwelling
- Automatic protection applies without filing
- Only one homestead per family
- Must reside in dwelling

### Limitations
- Does not protect against:
  - Mortgages and deeds of trust
  - Mechanic's liens
  - Property tax liens
  - Child support/alimony
  - Debts secured by dwelling before homestead

## Transfer on Death Deed
**Probate Code § 5600 et seq.** (effective 2016, extended 2021)

### Features
- Revocable transfer of real property on death
- Avoids probate for the property
- Does not affect Prop 13 reassessment until transfer
- Owner retains full control during lifetime

### Requirements
- Must use **statutory form** (Prob. Code § 5642)
- Must be **notarized**
- Must be **recorded within 60 days of signing**
- Revocable until death

### Limitations
- Subject to decedent's debts
- Medicaid estate recovery applies
- Subject to claims of creditors
- Does not avoid Prop 19 reassessment

### Revocation
- Must be recorded before death
- New TODD supersedes prior
- Conveyance during life revokes

## Probate Process

### Overview
- California probate is **expensive** and **slow**
- Fees based on **gross estate value** (not net)
- Can take **1-2+ years**
- Strong incentive to use revocable trusts

### Statutory Probate Fees (Prob. Code § 10810)
| Estate Value | Attorney Fee | Executor Fee |
|--------------|--------------|--------------|
| First $100K | 4% | 4% |
| Next $100K | 3% | 3% |
| Next $800K | 2% | 2% |
| Next $9M | 1% | 1% |
| Next $15M | 0.5% | 0.5% |
| Over $25M | Reasonable | Reasonable |

### Example
- $1 million gross estate = $23,000 attorney fee + $23,000 executor fee = $46,000 minimum

### Small Estate Procedures
- **Affidavit procedure**: Personal property under $184,500 (2025)
- **Spousal property petition**: Simplified for surviving spouse
- 40-day waiting period for affidavit

### Timeline
- Typical probate: 12-24 months
- Complex estates: 2-4+ years
- Creditor claim period: 4 months minimum

## Proposition 13 and 19 Implications

### Proposition 13 (1978)
- Property tax limited to 1% of assessed value
- 2% maximum annual increase
- Reassessment triggered by change in ownership

### Proposition 19 (2021)
- **Limits parent-child exclusion**
- Inherited primary residence:
  - Must be child's primary residence within 1 year
  - Value over $1 million above current assessed value reassessed
- Investment/rental property: No exclusion (fully reassessed)

### Planning Implications
- Inherited real estate may face significant tax increase
- Important to evaluate before inheriting
- Consider holding real estate in trust carefully
- May affect decision to sell vs. keep property

## Unique State Features

### High State Income Tax
- Top marginal rate over 13%
- Trust income taxed to California if California trustee or beneficiary
- Consider trust situs planning for income tax
- Important for high-income beneficiaries

### Creditor Protection
- Strong homestead exemption
- 401(k) and IRA protections (CCP § 704.115)
- Limited asset protection trust options
- California does not have DAPT statute

### Planning Considerations
- **Revocable trust essential** to avoid expensive probate
- Community property agreement/trust for proper characterization
- Transfer on death deed for real property outside trust
- Consider Prop 19 implications for inherited real estate
- Medi-Cal planning should account for unique rules
- High cost of care makes Medicaid planning important
- Step-up in basis on 100% of community property is major advantage
- Income tax planning critical due to high rates

## Safe Deposit Box Rules

### Access After Death (Prob. Code § 331)
- Joint renter has immediate access
- Without joint renter: letters testamentary from superior court required
- California Probate Code allows **limited access** with death certificate to search for:
  - Will
  - Trust
  - Burial instructions
- Inventory required; no removal without authority

### Joint Ownership
- Surviving joint renter has full access rights
- No automatic freeze upon death
- Contents accessible without probate
- Community property considerations apply to box contents

### Inventory Requirements
- Contents may need to be inventoried for estate purposes
- Community property vs. separate property characterization important
- Include box contents in trust funding

### Best Practices
- Original will should NOT be stored in safe deposit box
- Community property assets should be properly characterized
- Document box contents for estate administration
- Inform successor trustee/executor of box location
- Consider adding fiduciary as authorized signer
