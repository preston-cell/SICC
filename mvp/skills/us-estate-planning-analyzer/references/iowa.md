# Iowa

## Key Differences from Other States
- **No state inheritance tax** (fully repealed January 1, 2025)
- No state estate tax
- Common law property state
- **Unlimited rural homestead protection** (40 acres rural, 1/2 acre urban)
- Transfer on Death Deed available
- Iowa Trust Code adopted
- Strong spousal protections

## Will Requirements
**Statutory Basis**: Iowa Code Chapter 633 (Iowa Probate Code)

### Execution Requirements (Iowa Code § 633.279)
- Testator must be **18+ years old**
- Must be of **sound mind**
- Must be **in writing**
- Must be **signed by testator** (or by another at direction in presence)
- Must be **subscribed in presence of testator** by **2 competent witnesses**
- Witnesses must sign in presence of testator

### Self-Proving Affidavit (Iowa Code § 633.279)
- Available and recommended
- Testator and witnesses sign affidavit before notary
- Creates presumption of proper execution
- Avoids need to locate witnesses at probate
- Must be executed at same time as will

### Holographic Wills
- **NOT recognized** in Iowa
- All wills must be witnessed
- Foreign holographic wills recognized if valid where executed

### Witness Requirements
- Must be competent witnesses (understand nature of act)
- Interested witness does not invalidate will
- Should not be beneficiaries (best practice)
- Must sign in presence of testator

### Foreign Wills (Iowa Code § 633.278)
- Valid if executed in compliance with:
  - Iowa law, OR
  - Law of state/country where executed, OR
  - Law of testator's domicile at execution or death

## Trust Requirements
**Statutory Basis**: Iowa Code Chapter 633A (Iowa Trust Code)

### Trust Creation (Iowa Code § 633A.2102)
- Iowa Trust Code adopted (based on UTC)
- Must have trust property
- Must have ascertainable beneficiaries (or valid purpose)
- Settlor must have capacity
- Writing required for real property

### Trust Administration
- Iowa Trust Code governs
- Prudent Investor Rule applies
- Trustee has duties of loyalty, impartiality
- Accounting to beneficiaries required
- Notice requirements upon death

### Revocable Trusts (Iowa Code § 633A.3101)
- Common for probate avoidance
- Settlor retains control during lifetime
- Presumed revocable unless stated otherwise
- Becomes irrevocable at settlor's death
- Creditor claim period applies

### Directed Trusts
- Iowa allows directed trusts
- Trust advisor can direct trustee
- Divided trustee responsibilities permitted
- Trustee liability limited when following proper directions

### Trust Decanting (Iowa Code § 633A.4801)
- Authorized under Iowa Trust Code
- Allows modification of irrevocable trusts
- Requires trustee discretionary power
- Notice requirements apply

## Power of Attorney
**Statutory Basis**: Iowa Code Chapter 633B (Iowa Uniform Power of Attorney Act)

### Execution Requirements (Iowa Code § 633B.105)
- Must be **signed by principal** (or at direction in presence)
- Must be **notarized** (acknowledged)
- Witnesses recommended but not required
- Must clearly identify agent

### Durability (Iowa Code § 633B.104)
- Durable by default unless document states otherwise
- Non-durable POA terminates upon incapacity
- Springing POA available (effective upon incapacity)

### Statutory Form (Iowa Code § 633B.301)
- Iowa provides statutory form (recommended)
- Other forms valid if properly executed
- Third parties must accept valid POA

### Agent Duties (Iowa Code § 633B.114)
- Fiduciary duty to principal
- Must act in principal's best interest
- Must keep records of transactions
- Must avoid conflicts of interest

### Hot Powers (Iowa Code § 633B.201)
- Certain powers require express grant:
  - Create, amend, revoke trusts
  - Make gifts
  - Create/change beneficiary designations
  - Create/change rights of survivorship
  - Delegate authority

### Third Party Acceptance (Iowa Code § 633B.120)
- Third parties must accept valid POA
- May request agent certification
- Refusal may result in liability
- Good faith reliance protected

## Healthcare Directive
**Statutory Basis**: Iowa Code Chapter 144A (Iowa Life-Sustaining Procedures Act); Iowa Code Chapter 144B (Durable Power of Attorney for Health Care)

### Types Available
- Declaration (Living Will) - Iowa Code § 144A.3
- Durable Power of Attorney for Health Care - Iowa Code § 144B.2
- Combined Advance Directive

### Living Will Requirements (Iowa Code § 144A.3)
- Must be **in writing**
- Must be **signed by declarant** (or at direction)
- Must be **witnessed by 2 adults**
- Witnesses cannot be:
  - Related by blood, marriage, or adoption
  - Entitled to any portion of estate
  - Attending physician or employee
  - Employee of healthcare facility where declarant is patient
- Notarization recommended but not required

### Durable Power of Attorney for Health Care (Iowa Code § 144B.2)
- Must be **in writing**
- Must be **signed by principal** (or at direction)
- Must be **witnessed by 2 adults** OR **notarized**
- Witnesses cannot be agent or healthcare provider

### Agent Authority
- Make healthcare decisions when principal incapacitated
- Access medical records
- Consent to or refuse treatment
- Arrange care and placement
- Make anatomical gift decisions
- Cannot authorize commitment unless specifically granted

### Iowa POLST
- Available in Iowa
- Medical orders for seriously ill patients
- Completed with healthcare provider
- Portable across healthcare settings
- Pink form

### Default Surrogate (Iowa Code § 144A.7)
- If no healthcare directive:
  - Guardian
  - Spouse
  - Adult child
  - Parent
  - Adult sibling
  - Other relative
  - Close friend

## Estate and Inheritance Tax

### Iowa Inheritance Tax - REPEALED
- **Fully repealed** as of **January 1, 2025**
- Deaths on or after January 1, 2025 - NO inheritance tax
- Phase-out occurred 2021-2024

### Historical Context
- Previously had inheritance tax with rates up to 15%
- Tax based on relationship to decedent
- Class A (closest relatives) exempt
- Class B and C had graduated rates
- Completely eliminated effective 2025

### Current Status
- **No state estate tax**
- **No state inheritance tax**
- **No state gift tax**
- Only federal estate tax applies if estate exceeds federal threshold
- 2025 federal exemption: $13.99 million per person

## Medicaid Planning (Iowa Medicaid)

### Look-Back Period
- **60 months** (5 years)

### Asset Limits (2025)
- Individual: $2,000
- Married (community spouse): CSRA up to federal maximum (~$154,140)
- CSRA minimum: approximately $31,584

### Income Limits
- Approximately $2,829/month for individuals
- Qualified Income Trust (Miller Trust) available if over limit

### Exempt Assets
- Primary residence (equity limit ~$713,000)
- One vehicle
- Personal belongings and household goods
- Prepaid irrevocable burial contracts
- Term life insurance
- Whole life with face value under $1,500

### Iowa-Specific Programs
- **Iowa Medicaid** administered by Iowa HHS
- **HCBS (Home and Community Based Services) Waiver**
- **Elderly Waiver** program
- **PACE** programs in some areas
- Estate recovery program active

### Spousal Protections
- Community spouse resource allowance (CSRA)
- Minimum monthly maintenance needs allowance (MMMNA)
- Court order for increased CSRA available

### Penalty Divisor
- Approximately $7,500-8,500/month (verify current rate)

## Homestead Protection
**Iowa Code Chapter 561**

### Exemption Amount - One of Strongest in Nation
- **Rural**: Up to **40 acres** (unlimited value)
- **Urban**: Up to **1/2 acre** (unlimited value)
- No cap on dollar value
- Must contain dwelling house

### Requirements
- Must be **principal residence**
- Must be owned by person or spouse
- Only one homestead per family
- Automatic protection (no filing required)

### Scope of Protection
- Protected from most judgment creditors
- Does NOT protect against:
  - Mortgages and deeds of trust
  - Property tax liens
  - Mechanic's liens (for work on homestead)
  - Purchase money obligations

### Surviving Spouse Rights
- Surviving spouse has right to remain in homestead for life
- Descends to surviving spouse free from debts
- Strong protection for surviving spouse

### Setting Apart
- Homestead must be "set apart" before creditor claims
- Court process to establish boundaries
- Excess land above limits subject to creditor claims

## Transfer on Death Deed
**Iowa Code Chapter 558C**

### Features
- Transfer real property on death without probate
- Revocable during owner's lifetime
- Does not affect owner's rights during lifetime
- Simple alternative to trust for real property

### Requirements
- Must be in **writing**
- Must be **signed** by owner
- Must be **acknowledged (notarized)**
- Must be **recorded before death** with county recorder
- Must designate grantee-beneficiary

### Limitations
- Subject to decedent's debts
- Medicaid estate recovery applies
- Subject to claims of creditors
- Only for Iowa real property

### Revocation
- Can revoke by recording revocation instrument
- Subsequent TOD deed supersedes prior
- Conveyance during life revokes
- Not revoked by will

## Probate Procedures

### Types of Administration
- **Formal Administration** - Court supervised
- **Summary Administration** - For smaller estates
- **Informal Administration** - Limited court involvement

### Small Estate Procedures (Iowa Code § 633.356)
- **Summary administration** for estates where:
  - Costs exceed value, OR
  - Estate under $100,000 and all heirs consent
- Simplified procedures
- Reduced court involvement

### Timeline
- Typical probate: 6-12 months
- Complex estates: 12-24 months
- Small estates: 3-6 months
- Creditor claim period: 4 months from second publication

### Court Jurisdiction
- **District Court** handles probate
- Probate Court within District Court
- County where decedent resided has jurisdiction

### Executor/Personal Representative
- Named in will or appointed by court
- Bond may be required
- Can request unsupervised administration

## Unique State Features

### No Death Taxes (as of 2025)
- No state estate tax
- No state inheritance tax (fully repealed)
- Simplified estate administration
- Only federal considerations

### Unlimited Homestead Value
- One of strongest homestead protections
- 40 acres rural / 1/2 acre urban
- No dollar limit on exemption
- Survives to spouse

### Strong Spousal Rights
- Spousal share rules
- Elective share provisions
- Homestead rights
- Cannot fully disinherit spouse

### Agricultural Considerations
- Farm succession planning important
- Special agricultural provisions
- Family farm planning
- Conservation easements common

### Planning Considerations
- No state death tax simplifies planning
- TOD deeds effective for real property
- Unlimited homestead provides strong asset protection
- Iowa Trust Code provides modern trust administration
- Consider revocable trust for privacy
- Strong spousal protections must be considered

## Safe Deposit Box Rules

### Access After Death (Iowa Code § 633.88)
- Joint renter has immediate access
- Without joint renter: letters testamentary required
- Bank may permit examination with death certificate
- Can search for will, burial instructions
- Inventory and court authorization needed for removal

### Joint Ownership
- Surviving joint renter has full access rights
- No automatic freeze upon death
- Contents accessible without probate
- Consider liability exposure

### Examination for Will
- Financial institution may permit examination
- Must have death certificate
- Cannot remove contents without authority
- Inventory of contents required

### Best Practices
- Original will should NOT be stored in safe deposit box
- Document box contents with estate plan
- Inform executor of location
- Consider adding fiduciary as deputy

## State Charitable Incentives

### Endow Iowa Tax Credit
- 25% state tax credit for gifts to qualified community foundations
- Applies to gifts to permanent endowment funds
- Individual: maximum $300 credit ($1,200 gift)
- Married filing jointly: maximum $600 credit ($2,400 gift)
- Additional federal charitable deduction available
- Credits are first-come, first-served (apply early in tax year)

### School Tuition Organization Tax Credit
- 75% credit for contributions to school tuition organizations
- Supports private school scholarships
- Limited total credits available statewide
- Pre-approval required

### Estate Planning Implications
- Endow Iowa credit makes community foundation giving very efficient
- 25% state credit + federal deduction = significant tax benefit
- Limited credits available - coordinate timing of large gifts
- Consider establishing giving pattern during Iowa residency
- Community foundation gifts create permanent local legacy
